VERIFICATION
Verification is intended to assist the Chocolatey moderators and community
in verifying that this package's contents are trustworthy.

The binaries included in this package are downloaded from the official GitHub releases:
https://github.com/Polar-Team/diffusion/releases

The package downloads binaries directly from GitHub releases during installation,
ensuring that users always get the official, signed releases.

All releases are:
- Signed with Cosign (keyless)
- Include SLSA Level 3 provenance
- Have SHA256 checksums published

To verify the binaries manually:

1. Download the archive from: https://github.com/Polar-Team/diffusion/releases
2. Verify the checksum against SHA256SUMS file
3. Verify the Cosign signature:
   ```powershell
   cosign verify-blob `
     --certificate diffusion-windows-amd64.zip.pem `
     --signature diffusion-windows-amd64.zip.sig `
     --certificate-identity-regexp="https://github.com/Polar-Team/diffusion" `
     --certificate-oidc-issuer="https://token.actions.githubusercontent.com" `
     diffusion-windows-amd64.zip
   ```
